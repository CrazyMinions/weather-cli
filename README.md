# 🌤️ weather-cli

[中文版](README.zh.md)

A command-line weather query tool built with TypeScript.

## ✨ Features

- Query weather for any city worldwide
- Beautiful terminal output with colors and emojis
- Fast API responses using wttr.in
- Simple and intuitive command-line interface

## 📦 Installation

### Prerequisites

- Node.js (v20 or higher)
- npm

### Install globally

```bash
npm install -g .
```

Or clone and run locally:

```bash
git clone https://github.com/CrazyMinions/weather-cli.git
cd weather-cli
npm install
npm run start
```

## 🚀 Usage

### Query weather for a specific city

```bash
weather-cli Beijing
```

### Query weather for default city (Beijing)

```bash
weather-cli
```

### Example output

```
🌤️  Weather Report
Location: Beijing
Temperature: 25°C (Feels like 27°C)
Condition: Sunny
Humidity: 45%
Wind: 12 km/h
UV Index: 6
```
